# 🚖 Uber Power BI Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze Uber ride booking data. The dashboard provides insights into booking performance, revenue, customer behavior, vehicle utilization, ride cancellations, and busy locations.

The project demonstrates Power BI skills including Data Modeling, Power Query, DAX Measures, Interactive Navigation, and Dashboard Design.
